Политика конфиденциальности

Приложение: Problem? | Дата обновления: 9 июня 2026

1. Назначение приложения

Problem? - приложение для локального дневника самоанализа. Пользователь может создавать записи о ситуациях, мыслях, телесных ощущениях, последствиях и желаемых действиях без проблемы.

2. Какие данные обрабатываются

Пользователь самостоятельно вводит текстовые записи внутри приложения. Эти записи могут содержать любую информацию, которую пользователь решил сохранить в своем дневнике.

3. Где хранятся данные

Все записи хранятся локально на устройстве пользователя. Приложение не использует серверную часть, аккаунты, облачную синхронизацию, рекламу или аналитические SDK.

4. Передача данных

Приложение не передает записи разработчику и третьим лицам. Функция "Скопировать для AI" только помещает подготовленный текст в буфер обмена устройства. Пользователь самостоятельно решает, куда вставить этот текст и отправлять ли его во внешние сервисы.

5. Разрешения устройства

Текущая версия приложения не запрашивает доступ к камере, микрофону, геолокации, контактам, SMS, телефонным функциям, файлам пользователя или уведомлениям.

6. Удаление данных

Пользователь может удалять отдельные записи внутри приложения. При удалении приложения локальные данные могут быть удалены операционной системой устройства.

7. Медицинский статус

Problem? не является медицинским изделием, не оказывает медицинские услуги, не ставит диагнозы и не заменяет психолога, психотерапевта, врача или кризисную помощь. При угрозе жизни, здоровью или безопасности необходимо обратиться за срочной профессиональной помощью или в экстренные службы.

8. Контакты

По вопросам конфиденциальности напишите на draazaripov@yandex.ru.

Privacy Policy

App: Problem? | Last updated: June 9, 2026

1. App purpose

Problem? is a local self-reflection diary app. Users can create entries about situations, thoughts, body sensations, consequences, and what they would do without the problem.

2. Data processed by the app

Users manually enter text notes inside the app. These notes may contain any information the user chooses to save in their diary.

3. Where data is stored

All diary entries are stored locally on the user's device. The app does not use a server backend, accounts, cloud sync, advertising, or analytics SDKs.

4. Data sharing

The app does not transmit diary entries to the developer or third parties. The "Copy for AI" feature only copies prepared text to the device clipboard. The user decides where to paste that text and whether to send it to any external service.

5. Device permissions

The current version of the app does not request access to the camera, microphone, location, contacts, SMS, phone features, user files, or notifications.

6. Data deletion

Users can delete individual entries inside the app. When the app is removed, local data may be deleted by the device operating system.

7. Medical status

Problem? is not a medical device, does not provide medical services, does not diagnose conditions, and does not replace a psychologist, psychotherapist, doctor, or crisis support. If there is a threat to life, health, or safety, seek urgent professional help or contact local emergency services.

8. Contact

For privacy questions, contact draazaripov@yandex.ru.
